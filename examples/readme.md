# DecLearn Examples

Here you can find directories containing examples and tutorials of how you can
use, deploy or adapt DecLearn.


- `adding_rmsprop` : Example of how to write a custom `Optimizer` module
(`OptiModule`) using DecLearn abstractions.

- `flamby` : Example of how to use DecLearn to run a federated learning
experiment on a dataset coming from the FLamby dataset playground.

- `heart-uci` : Example of how to use DecLearn to run a federated learning
experiment on a heart disease dataset from the UCI ML repository.

- `mnist` : Example of how to use DecLearn to run a federated learning
experiment on the MNIST dataset.

- `mnist_g5k` : Example of how to deploy DecLearn on Grid5000 to run a federated
learning experiment on the MNIST dataset.

- `mnist_quickrun` : Example of how to use the `declearn-quickrun` tool to
easily run a simulated federated learning experiment on the MNIST dataset.

- `nlp` : Example of how to use DecLearn to run a federated learning experiment
that trains a transformer model retrieved from HuggingFace on the IMDb dataset.
